Detailed Description

Interface class representing two-dimensional object with index [channel_id, field_name + (type)].

e.g: +—+-----—+------—+------—+------—+ |#ch|const |vtx_x |vtx_y |vtx_z | +—+-----—+------—+------—+------—+ |0 |2.000000|11.000000|12.000000|13.000000| +—+-----—+------—+------—+------—+
